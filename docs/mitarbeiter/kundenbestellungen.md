# Kundenbestellungen

Wünscht ein Kunde einen Artikel, den Sie gerade nicht vorrätig haben, erfassen Sie eine **Kundenbestellung** (Sonderbestellung). WorkTime hält Wunsch, Kunde und Status zusammen, damit nichts vergessen wird.

## Abgrenzung: Kundenbestellung vs. Einkauf

- Eine **Einkaufsbestellung** richten Sie an einen **Lieferanten**, um Ihren Bestand aufzufüllen – siehe [Bestellungen & Wareneingang](article:bestellungen-einkauf).
- Eine **Kundenbestellung** hält den Wunsch eines **Kunden** fest, den Sie besorgen und dann übergeben.

Beides kann zusammenhängen (Sie bestellen die Kundenware beim Lieferanten mit), ist in der App aber getrennt geführt.

## Eine Kundenbestellung erfassen

1. Im **Laden**-Bereich öffnen Sie die Kundenbestellungen.
2. Wählen oder erfassen Sie den **Kunden** (aus den [Kontakten](article:kontakte)).
3. Legen Sie die gewünschten **Artikel/Positionen** in den Warenkorb.
4. Speichern Sie die Bestellung.

## Status verfolgen

Die Bestellung hat einen **Status** – von der Aufnahme über „ist da" bis „abgeholt/geliefert". So sieht jeder im Team, was noch offen ist und welcher Kunde benachrichtigt werden kann.

> [!TIP]
> Hinterlegen Sie beim Kunden eine Telefonnummer oder Notiz. Dann kann jede Kollegin ihn erreichen, sobald die Ware eingetroffen ist – auch wenn sie die Bestellung nicht selbst aufgenommen hat.

## Übergabe an den Kunden

Ist die Ware da und der Kunde informiert, schließen Sie die Bestellung bei Abholung/Lieferung ab. Damit ist der Vorgang sauber dokumentiert.

> [!NOTE]
> Halten Sie Sonderwünsche (Menge, Variante, Termin) genau fest. Missverständnisse bei Sonderbestellungen führen sonst zu Fehlkäufen.
